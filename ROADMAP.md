# Termify Roadmap

Welcome to the Termify roadmap! This document outlines our vision and planned development phases for creating the ultimate ultra-light, smart SSH client.

## 🚀 Phase 1: Core Essentials (Stable)
Focus: Establishing the fundamental experience of a high-performance terminal.
- [x] **Terminal Framework**: High-performance rendering for lightning-fast input/output.
- [x] **Session Management**: Easy organization of remote connections with metadata.
- [x] **Authentication Engine**: Secure storage of credentials and integration with SSH-Agent.
- [x] **Dark Mode & Aesthetics**: A clean, focused UI designed for modern developers.

## 🛡️ Phase 2: Enhanced Security (In Progress)
Focus: Strengthening your data and simplifying complex secure workflows.
- [/] **Encrypted Vaults**: Next-gen storage for keys and configurations using industry-standard AES-256.
- [ ] **Native SSH Key Management**: Streamlined UI for generating and deploying SSH keys.
- [ ] **Multi-factor Authentication (MFA)**: Support for second-factor authentication for SSH sessions.
- [ ] **Granular Access Control**: For teams sharing vaults and sensitive configuration.

## ⚡ Phase 3: Productivity & Teamwork (Planned)
Focus: Making your workflow faster and collaborative.
- [ ] **Snippet Library**: Save and sync common command snippets across sessions.
- [ ] **Team Sharing**: Seamlessly share secure vaults with your team members.
- [ ] **SFTP/SCP Integration**: Simple, integrated file browser for direct remote file operations.
- [ ] **Enhanced Autocomplete**: Context-aware command suggestions tailored to your environment.

## 🏗️ Phase 4: Advanced Integration (Future)
Focus: Pushing the boundaries of what an SSH client can do.
- [ ] **AI-Assisted Commands**: Intelligent suggestions for command syntax and common tasks.
- [ ] **Plugin System**: Build and share custom plugins to extend Termify's functionality.
- [ ] **Cross-Platform Sync**: Synchronize your workspace across different machines and devices.
- [ ] **Mobile Support**: Access your sessions on the go via a high-performance mobile client.

---
> [!NOTE]
> This roadmap reflects our current goals and is subject to change based on community feedback. Have an idea? Feel free to [submit a feature request](https://github.com/TermifyTeam/Termify/issues/new?template=feature_request.md)!
